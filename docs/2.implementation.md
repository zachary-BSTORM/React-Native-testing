
# Tutoriel : Application de gestion de tâches avec React Native, Expo Router et Context

Ce tutoriel vous guide dans la création d'une application mobile de gestion de tâches (todo) avec React Native, Expo Router et le Context API. L'application permet d'ajouter, supprimer, éditer et marquer des tâches comme terminées.

## Fonctionnalités principales
- Ajouter une tâche avec un titre et une description
- Supprimer une tâche
- Marquer une tâche comme terminée ou non terminée (toggle)
- Modifier le titre et la description d'une tâche
- Navigation entre la liste et la page de détails grâce à Expo Router

## Structure des pages

### 1. Page principale (Liste des tâches)
- Affiche toutes les tâches sous forme de liste
- Champ de saisie pour le titre et la description d'une nouvelle tâche
- Bouton pour ajouter la tâche
- Chaque tâche affiche :
	- Son titre (barré si terminée)
	- Un bouton pour marquer comme terminée ou annuler
	- Un bouton pour supprimer
	- Un clic sur la tâche permet d'accéder à la page de détails

#### Fonctions utilisées
- `addTodo(todo)` : ajoute une tâche
- `removeTodo(id)` : supprime une tâche
- `toggleTodo(id)` : change le statut terminée/non terminée
- Navigation : `router.push('/details/[id]')` ou `/details?id=...`

### 2. Page de détails d'une tâche
- Affiche le titre et la description de la tâche sélectionnée
- Champs éditables pour modifier le titre et la description (état local)
- Bouton pour enregistrer les modifications (remplace la tâche dans le contexte)
- Bouton pour revenir à la liste après modification
- Statut cliquable pour inverser l'état terminée/non terminée

#### Fonctions utilisées
- `toggleTodo(id)` : change le statut
- `removeTodo(id)` puis `addTodo(todo modifié)` : met à jour la tâche
- Navigation : `router.replace('/')` pour revenir à la liste

## Architecture technique
- Utilisation du Context API pour gérer l'état global des tâches (`TodoProvider`)
- Navigation gérée par Expo Router (`_layout.tsx` avec `<Stack />`)
- Composants React Native (`View`, `Text`, `TextInput`, `TouchableOpacity`, `Pressable`, etc.)

## Exemple de navigation
Dans la liste :
```tsx
<Pressable onPress={() => router.push(`/details/${item.id}`)}>
	<Text>{item.title}</Text>
</Pressable>
```
Dans la page de détails, pour sauvegarder :
```tsx
<TouchableOpacity onPress={handleSave}>
	<Text>Enregistrer et revenir à la liste</Text>
</TouchableOpacity>
```
